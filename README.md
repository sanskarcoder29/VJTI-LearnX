# VJTI LearnX

VJTI LearnX is a student-focused platform designed to make study resources easily accessible for VJTI students.

Instead of searching through WhatsApp groups or scattered Google Drive links, students can find organized notes, PDFs, and academic materials for different semesters in one place.

## 🌐 Live Website
https://vjti-learn-x.vercel.app/

## 🚀 Features
- Organized study materials
- Easy access to semester-wise resources
- Simple and clean user interface
- Built for VJTI students

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
The goal of this project is to simplify how students access academic resources and create a centralized hub for learning materials.
